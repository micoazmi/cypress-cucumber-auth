Navigate to the `app/NextJs/firebase-auth` directory and install its dependencies.

```bash

npm install
npm run dev
```

Navigate to the `app/NextJs/firebase-auth` directory and install its dependencies.

```bash
cd cypress-cucumber

npx cypress open
```
